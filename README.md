# Globant — Agentic Banking for the GCC

A self-contained, body-only-embeddable landing page for Globant's GCC banking & financial-services
campaign, themed around the strategic shift **from digital to agentic banking**.

## Highlights

- **Single self-contained `index.html`** — no build step, no framework.
- **Brand-accurate** — Heebo + Open Sans (self-hosted), the Globant navy/lime/turquoise system, and
  the Globant arrow as a recurring signature device.
- **Vector-only imagery** — 100% inline SVG; no raster or AI-generated images.
- **Scroll-driven motion** — GSAP + ScrollTrigger (self-hosted), reduced-motion safe, and the page
  renders fully without JavaScript.
- **Embed-ready** — all CSS/JS namespaced `gb-lp-`, body-only content block, GTM/Pardot friendly.

## Preview

Open `index.html` in any modern browser.

## Structure

```
index.html          # the page (self-contained)
assets/fonts/       # self-hosted Heebo + Open Sans (.woff2)
assets/logos/       # Globant + ecosystem marks (SVG)
assets/vendor/      # GSAP + ScrollTrigger
```

## Sections

Hero → the shift → three pillars → wealth management (GCC) → the proof → how Globant delivers (AI Pods)
→ gated report (Smart Banking 2026) → final CTA.
